# Error Log Index

Central index for project errors, failures, and resolutions.

| ID | Date | Component | Summary | Status | Log File |
|----|------|-----------|---------|--------|----------|
| — | — | — | No errors logged yet | — | — |

## How to Log an Error

1. Create a new file: `docs/error-logs/YYYY-MM-DD-short-description.md`
2. Include: date, component, error message, steps to reproduce, resolution (or open)
3. Add a row to this index table

## Template

```markdown
# Error: [Short Title]

- **Date:**
- **Component:** backend | frontend | firmware | hardware
- **Severity:** low | medium | high | critical

## Description

## Steps to Reproduce

## Expected vs Actual

## Resolution

## Prevention
```
