# Firmware Examples

Beginner-friendly sketches for first hardware bring-up.

| Example | Purpose |
|---------|---------|
| `basic_serial_test/` | First upload after boards arrive — Serial counter only |
| `wifi_connection_template/` | Safe Wi-Fi template with placeholders (no real credentials) |

These examples are **templates**, not production firmware.

Open each folder's `.ino` file in Arduino IDE when hardware is available.
