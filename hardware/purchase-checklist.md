# Purchase Checklist

Use this checklist when ordering prototype hardware.

## Microcontrollers & Camera

- [ ] ESP32 DevKit (×1)
- [ ] ESP32-S3 with camera support (×1)
- [ ] OV2640 camera module (×1)

## Sensors

- [ ] HC-SR04 ultrasonic (×2)
- [ ] IR obstacle / break-beam pair (×2)
- [ ] SW-420 vibration module (×1)

## Output

- [ ] 5V active buzzer (×1)
- [ ] 5V relay module (×1)
- [ ] LEDs green / amber / red (×1 each)

## Prototyping Supplies

- [ ] Breadboards (×2)
- [ ] Jumper wire kit
- [ ] 5V 3A power supplies (×2)
- [ ] USB cables for programming (×2)

## Enclosure & Field (Later)

- [ ] Weatherproof enclosure (×2)
- [ ] Cable glands
- [ ] Mounting brackets for camera

## Notes

- Order spare sensors for failure replacement
- Prefer reputable suppliers for ESP32-S3 (camera timing sensitive)
- Keep receipts for project documentation
